Meteor Shower is a game about a bouncing planet avoiding meteors which move unpredictably

It was made in Visual Studio 2019 and uses the SFML library

The game is in x64->Release->Meteor Shower.exe

Controls:
Left arrow key - move left
Right arrow key - move right
ESC - close the game

The game will start immediately, be ready
